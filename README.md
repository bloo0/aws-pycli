aws-pycli
==============================

Manage your AWS Services via Python

Getting Started
------------

1. Install the following python module:
- `pip install click`
- `pip install python-dotenv`
- `pip install termcolor`

2. Setup AWS Assume Role
3. Add your AWS Regions
4. Set your environmental variables
